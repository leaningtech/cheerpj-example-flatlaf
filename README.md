# CheerpJ example: FlatLaf Look and Feel Demo

This example shows how to use CheerpJ to run the FlatLaf Look and Feel Demo with Java 17 in the browser.

---

To run this example locally:

```sh
git clone https://github.com/leaningtech/cheerpj-example-flatlaf.git
cd cheerpj-example-flatlaf
npx http-server
```
